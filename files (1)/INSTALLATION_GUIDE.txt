QUICK SETUP (No Download Needed)
=================================================================

Option A: Run Everything in Docker (Easiest)
--------------------------------------------
1. Download docker-compose.yml from outputs
2. Create project folder: mkdir ecosphere && cd ecosphere
3. Paste docker-compose.yml content into file
4. Run: docker-compose up --build
5. Access: http://localhost:3000

Option B: Manual Setup
----------------------
Backend:
  1. Create backend folder
  2. Paste server.js
  3. Paste package.json
  4. Create prisma folder and paste schema.prisma, seed.js
  5. Run:
     npm install
     npm run dev

Frontend:
  1. Create frontend folder  
  2. Paste package.json, next.config.js
  3. Create app folder and paste all page.tsx files
  4. Paste globals.css
  5. Run:
     npm install
     npm run dev
